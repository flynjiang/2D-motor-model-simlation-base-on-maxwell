   2D-motor-model-simulation-base-on-maxwell

  基于 ANSYS Maxwell 的二维永磁同步电机建模与电磁仿真

 项目简介
本项目为电机学课程设计作业，使用 ANSYS Maxwell 有限元仿真软件，完成了一台永磁同步电机的二维几何建模、材料定义、激励加载与电磁场求解。通过建模仿真，深入理解了电机内部电磁场的分布规律及转矩产生机理。


电机设计计算：根据设计指标完成电机主要尺寸、绕组参数、磁路结构的计算
二维几何建模：在 Maxwell 2D 中绘制电机定转子冲片、绕组、永磁体及气隙等结构
材料属性赋值：为各部件指定硅钢片、铜导线、钕铁硼永磁体等材料参数
激励与边界设置：配置电流源激励、绕组连接方式、主从边界条件与网格剖分
求解与分析：进行静磁场/瞬态场求解，提取气隙磁密、反电动势、输出转矩等关键性能曲线
实验报告撰写：整理仿真流程与结果分析
